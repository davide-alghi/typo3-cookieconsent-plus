============
Contributing
============

Contributions are very welcome. Code, documentation, issues, new features.


Issues
======

If run into problem in code or documentation, you can open a new
`issue <https://github.com/davide-alghi/typo3-cookieconsent-plus/issues>`_.


Pull Requests
=============

Pull request is welcome: if you managed to fix a bug/issue, please create
a new `pull request <https://github.com/davide-alghi/typo3-cookieconsent-plus/pulls>`_.


New Features
============

To propose a new feature, please open a
`discussion <https://github.com/davide-alghi/typo3-cookieconsent-plus/issues>`_.

I'll be happy to talk about it.
